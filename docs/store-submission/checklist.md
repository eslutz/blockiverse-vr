# Store Submission Checklist

- App metadata
- Short description
- Long description
- Screenshots
- Trailer or capture, if available
- Comfort rating notes
- Privacy policy
- Data usage declarations
- Age and child-safety review
- VRC checklist
- Performance evidence
- Content checklist
- Store artwork
- Support email or site
- Known issues
- Release notes
- Signed release APK from a `main` tag
