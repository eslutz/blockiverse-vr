# Notice

Blockiverse VR is an original voxel sandbox project.

The project may be visually inspired by broad voxel sandbox conventions, but it must not copy Minecraft's protected identity. Do not commit or use Minecraft names, logos, screenshots, textures, UI, music, sounds, mob names, character names, fonts, item names used as distinctive branding, or characters visually derived from protected Minecraft characters.

All original project assets, including generated art and audio, must include provenance notes in `docs/art-direction/prompt-log.md` or a nearby source file when practical.
